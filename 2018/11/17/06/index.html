谷歌云免费搭建一年SS翻墙并有终身免费服务哦！

创建服务器


注册：https://cloud.google.com/ 打开网址进行平台服务的注册，必要的基本资料填写请按照提示进行，这个部分会要求输入信用卡的卡号，有效期等等。提交这个页面后，你的手机应该会收到提醒你的信用卡账户消费 1 美金，不过不用担心，这里是谷歌为了确认你不是机器人用于验证账户的，然后 5 分钟内你会收到账户撤销 1 美金的消费操作，即这 1 美元会返回你的账户，整个过程完全免费。而且你也不用担心，除非你主动确认升级成付费账号，300 美金的赠款消费完毕以后并不会直接从你的信用卡账户续费扣款。


美区的 micro 实例可以享受一年后继续免费的服务！



账户申请好以后，进入到控制台，在左边的菜单页中找到结算，点击来确认谷歌的赠送的 300 美金是否已经到账。



如下图显示，即为赠款已经到账，我们可以正式的开始进行超级上网的搭建环节了。



搭建开始（重点）


首先，开始创建VM实例（你不用懂这是什么意思）。位置在左上角的菜单页中的「computer engine」→「VM实例」。



进入后点击创建实例。



实例内的表单请按照下图所示进行填写，标红部分如图，没标注的默认即可。




创建成功就会出现一个VM实例的列表，如下图，然后在名称末尾的连接选项中选择「在浏览器窗口中打开」，之后正式进入搭建的配置界面




配置服务器



备注：这个部分的内容对于没有接触过编程的小伙伴来说，请严格按照给到的代码复制粘贴，然后按照提示一路回车基本就 OK 了，不用担心


sudo -i


wget —no–check–certificate https://freed.ga/github/shadowsocksR.sh; bash shadowsocksR.sh


如提示


wget :command not found


请执行


yum install wget –y


然后按提示操作，到这里一个能翻墙的服务器就搭建完成了，大家下载SSR客户端连接就能翻墙


配置防火墙规则


返回控制面板首页，选VPC网络，防火墙规则，


打开服务相关端口，入站创建一条防火墙规则，来源IP里写0.0.0.0/0，允许所有IP和端口访问。


不清楚的按照我下图设置。




完成


登陆客户端开始上网


因为下面你要用到本文前面给到的各平台的客户端了，在不同的平台的客户端中，填写好你这里的服务器信息，然后就可以愉快的超级上网了。这边仅以 Mac 端的配置举例进行说明。


打开客户端，找到服务器设置选项。



图中标注部分需要填写，填写的信息来自你自己的设置，说明见下文。



地址：请填写你的账户中 VM 实例页面中的外部 IP 的那一串数字



端口：请填写你之前自己设置的那串数字

加密方式：请选择而你之前的默认

密码：自然是你自己设置好的密码


以上的内容填写成功以后，点击确认。


在客户端中选择打开 shadowsocks开始上网
